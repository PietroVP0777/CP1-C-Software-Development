## CP1-C-Software-Development
Este projeto consiste em uma calculadora desenvolvida em C#, capaz de realizar as quatro operações matemáticas básicas:

- Adição
- Subtração
- Multiplicação
- Divisão

O programa utiliza estruturas de controle de fluxo como `while`, `switch` e `if/else`, conforme solicitado nas instruções do trabalho.

---

## 📋 Menu do Programa

Ao executar, o programa apresenta o seguinte menu:

<img width="1110" height="625" alt="image" src="https://github.com/user-attachments/assets/49bcd8ae-c455-4a6e-be00-14c7b6ecb969" />

O usuário escolhe uma opção digitando o número correspondente.

---

## ⚙️ Funcionalidades

✅ Solicita dois números após a escolha da operação  
✅ Realiza o cálculo corretamente  
✅ Exibe o resultado na tela  
✅ Continua executando até o usuário escolher sair  
✅ Trata erro de divisão por zero  
✅ Trata erro caso o usuário digite uma opção inválida no menu
